Web con Bootstrap

Desarrollo de una web para una empresa de eventos con Bootstrap, consta de un slider y una barra de navegacions con botones que abren ventanas modales. Es una web responsive y los botones se minimizan en un boton tipo "hamburguesa".

Animaciones con CSS

Animacion del movimiento de rebote de una pelota con CSS. Simulacion movimiento de rebote y de la gravedad de una pelota mediante código

Validacion formularios JavaScript

Validacion de formularios con JavaScrip para una web de registro de usuarios. El fomulario devuelve informacion cuando se completa si es erroneo y si escorrecta la validación.
